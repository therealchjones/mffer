Welcome.

mffer is designed to be run as a web app on the Google Cloud Platform. To deploy the application as a project of your own, you must register it as an Oauth 2.0 client, add your own API key, and set up a Google Sheets database.

If you just want to use the web app, you don't need to do this: just visit mffer.org.

If you want to run, build, or modify the command-line mffer program, which is used for extracting data and reporting it in a way the web app can use, you may not need to do this: visit mfferon GitHub.
